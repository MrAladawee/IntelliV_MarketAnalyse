# **IntelliV_MarketAnalyse**

Для корректной работоспособности необходимы следующие файлы:

_config.py_

_MODEL_NAME.pth_

_DeepPavlov_rubert model local_

_scaler_X.pkl
scaler_Y.pkl_

_PHONE.session_

## Описание

**1. data_ticker** - содержит датасет по обрабатываемым тикерам. В него входит набор *.csv файлов со следующими фичами:

| TRADEDATE  | CLOSE |
| ------------- | ------------- |
| YY-mm-dd  |  FLOAT  |

где TRADEDATE - дата торговой сессии; CLOSE - стоимость закрытия актива

**2. photo_ticker** - изображение логотипа тикера, который отправляется в сообщении телеграм-ботом

**3. prod_data, prod_data_MT** - данные, собранные за время работы модели (получение новостей + их обработка)

**4. img.jpg** - заглушка, если не будет необходимой фотографии в photo_ticker

**5. test.py** - попытка реализации парсинга телеграм канала MarketTwist (неудачно, или удачно, не помню)

**6. BotPred.py** - рабочий скрипт на парсинг со SmartLab

**7. launcher.py** - запуск двух сессий парсинга: MarketTwist; SmartLab + запуск общего парсинга цен в 03:00 за предыдущие торговые сессии, которых нет в датасете

**8. model.ipynb** - обучение самой модели + её апробация

**9. parse_news.ipynb** - сбор датасета с фильтрацией

**10. requirements.txt** - необходимые библиотеки проекта
